# North Dev — Site institucional

Site institucional mobile first desenvolvido com React, Vite, TypeScript e Tailwind CSS v4.

## Executar localmente

```bash
npm install
npm run dev
```

O endereço local será exibido no terminal pelo Vite.

## Validar a produção

```bash
npm run build
```

## Conteúdo

- Textos e dados do site: `src/data/site.ts`
- Miniaturas dos projetos: `public/projects`
- Materiais originais preservados: `site-content-reference`
